= Support

To get relatively quick access to help, **Discord** is where the chat community for SickRock is. This project is a free community open source project, and it relies on volunteers to spare their free time to help you. Please be patient and polite.

link:https://discord.gg/jhYWWpNJ3v[Chat on Discord]

If nobody is online, or you're not getting the right level of support, you can raise a ticket with the project's developers on GitHub. Again, please be patient and polite.

link:https://github.com/jamesread/SickRock/issues/new?assignees=&labels=support&template=support_request.md&title=[Open a support request on GitHub]
